# v0.7-v1.0 Closeout - Civilization Ecology Loop

## Status

v0.7 through v1.0 establish the first player-facing civilization layer on top of the existing observable simulation loop.

The core design pivot is complete: the player now makes understandable civilization decisions, while the lower-level simulation translates those decisions into ecological, resource, settlement, and replay effects.

## Added

- `CivilizationSystem` for settlements, civic decisions, seasonal climate, land use, soil fertility, forest cover, water table, flood risk, desertification, food, water, morale, health, and technology.
- Player-facing civic decisions:
  - `stewardship`
  - `add_population`
  - `expand_farms`
  - `clear_forest`
  - `rotate_crops`
  - `ration_food`
  - `manage_water`
  - `build_reservoir`
  - `reforest`
  - `research_irrigation`
  - `research_sanitation`
  - `research_crop_rotation`
  - `prepare_drought`
- Application flags for `--decision`, `--civic`, `--settlement`, and `--population`.
- Civilization events for settlement founding, settlement updates, decisions, seasons, land-use changes, and technology progress.
- Snapshot serialization for civilization aggregates and settlement projections.
- Terminal observer rendering for settlement markers and civic status.
- v1.0 integration test covering decisions, land impact, settlement viability, observer output, and replay inspection.
- v1.0 verification wrappers for shell and PowerShell.

## Verification

Primary closeout commands:

```sh
./scripts/verify_v1_0.sh
```

```powershell
.\scripts\verify_v1_0.ps1
```

The verifier builds the executable, runs default and 100-tick loops, runs a 24-tick observed civilization loop, checks missing-config failure behavior, then runs unit, integration, civilization, and legacy smoke tests.

## Gameplay Meaning

This version does not yet implement a full interactive UI. It does establish the underlying gameplay contract:

- The player acts through civilization choices.
- Choices affect settlement survival and land health.
- Natural pressures such as rainfall, drought, flood risk, soil depletion, forest cover, and desertification feed back into outcomes.
- The consequences are visible through snapshots, terminal observation, and replay inspection.
